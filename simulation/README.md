# Simulation
sim01、sim02、sim05、sim09については、bottom heavy性によるトルクを追加する前の状態のkapselを用いてシミュレーションを行っている。
その他については、bottom heavy性によるトルクの導入した後にシミュレーションを行っている。

# sim00
Bottom heavinessを付加することで、粒子の初期進行方向によらず、最終的に鉛直上向きに固定されることを確かめた。
```

```
